# Instructions for this "PROTFOLIO" project
## Source of truth
- 如果修改的代码与模版代码冲突，先问我再改。
- 修改代码前，将需要改动的地方为我总结，说出修改前与修改后的功能对比。

## 目标与要求
1. 找到我需要修改的地方，然后我手动将模版内容改为个人主页内容。
2. 修改完成后我们启动项目，以检查是否达到要求。
3. 检查达到我的要求后，再继续尝试添加功能：在READ RESUME部分添加下载RESUME功能。
4. 第3步完成后，再继续尝试添加功能：加入一个ai chatbot，用openai API，PostgreSQL，FASTapi进行搭建。

## 限制
- 完成第1步与第2步时，不允许修改项目的逻辑代码，不允许导入新的依赖。
- 完成第3步与第4步时，可以根据添加的功能修改需要的代码，允许导入新的依赖。但不允许修改我已经编辑好的个人主页内容。
